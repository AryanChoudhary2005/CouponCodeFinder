# Perk Hunter – AI-Powered Coupon & Affiliate Link Discovery Platform

Developed **Perk Hunter**, a web-based platform that helps users discover the latest discounted affiliate links and promotional offers for their favorite brands by extracting publicly available links from YouTube video descriptions. The platform automates the process of finding valid creator-sponsored deals, allowing users to save time and money while shopping online.

### Key Features

* Search brand-specific affiliate links (e.g., Byju's, Hostinger, NordVPN, etc.)
* Automatically extracts promotional links from YouTube video descriptions
* Filters recently uploaded videos to prioritize active and non-expired offers
* Displays direct discount and referral links in a clean, user-friendly interface
* Responsive frontend with category-based navigation and search functionality

### Technologies Used

**HTML, CSS, JavaScript, YouTube Data API v3, Axios, Vercel, REST APIs**

### Project Highlights

* Integrated the **YouTube Data API v3** to search videos based on user-entered brand keywords.
* Retrieved and analyzed video descriptions to automatically extract affiliate and promotional URLs.
* Implemented filtering based on upload date (recent videos) to improve the chances of finding active coupon codes and valid discount links.
* Filtered videos using engagement metrics such as views and likes to prioritize trusted and popular creator content.
* Built URL extraction and validation logic to identify genuine promotional links while excluding unrelated social media and external links.
* Designed a responsive and intuitive user interface with category browsing, search functionality, and deal discovery features.

### Workflow

1. User enters a brand name in the search bar.
2. The application searches YouTube for recent videos related to that brand.
3. Video descriptions are scanned to extract affiliate and promotional links.
4. URLs are filtered and validated to remove irrelevant links.
5. Users receive a list of recent discount links and promotional offers for the searched brand.

### Outcome

Perk Hunter demonstrates the practical use of APIs, web scraping techniques on publicly available metadata, URL processing, and frontend development to create a real-world solution that helps users discover the latest creator-sponsored discounts and promotional offers more efficiently.
